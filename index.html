fun colorToRGB(red: Int, green: Int, blue: Int): Int {
    var newPixel = 0
    newPixel = newPixel shl 8
    newPixel += red
    newPixel = newPixel shl 8
    newPixel += green
    newPixel shl 8
    newPixel += blue
    return newPixel

由题意大致知道在文件里写了信息，找个png图片,在图片末尾发现不正常字段，大部都是×*X *X,base64 转码，得知图片里9个二纬码条。

观察二维码得，二维码满足3个定位点+ 定位点四周为空白通道。将获得的9个二维码条，取他的蓝色通道，然后根据特征，推测出其中5张图片位置。剩下的使用google的zxing做排列组合，自动扫码得唯一密文。

下面是错误时间

密文，结合密钥加偏移索引，由于我是直接找到的密文，导致我丢失了关键信息

附加信息以asn.1 存储，同时指明了密钥，偏移量所在位置。

我直接将时间戳拆成了密钥+偏移量。然后一顿转换想得到能识别的信息，都已失败告终。最后发现二维码描述了s盒变化，执行s盒变化，仍是失败得不到有用信息。第二轮败北。

下面是正确时间

从文件头获取base64的comment,得到附加数据加密和存储位置，用asn.1 转换，得到解码要求所有信息，
